# Nottara — Marketing Website UI Kit

## Files
- `index.html` — single-page assembly: nav + hero + features + quote + pricing + footer.
- `components.jsx` — `NWNav`, `NWHero`, `NWFeatures`, `NWQuote`, `NWPricing`, `NWFooter`, plus `NWLogo` and `NWMark`.

## Sections covered
| Section | Notes |
|---|---|
| Nav | Sticky off-white with primary amber CTA pill |
| Hero | Display serif H1, secondary outline button, supporting visual mock |
| Features | 4 cards on 2-column grid |
| Quote | Inverted graphite section with brand mark |
| Pricing | 2 tiers, soft-amber emphasis card |
| Footer | Graphite, columns, legal row |
